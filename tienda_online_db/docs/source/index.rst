Proyecto Final Tienda Online
======================

Documentación técnica del proyecto final desarrollado para el curso de **Gestión de Bases de Datos (GBD)** en **Tokio School**, impartido por **David Morán**.

Autor: **Pablo Techera**
Año académico: **2025**

Descripción general
-------------------

Este proyecto consiste en la construcción de un sistema completo de gestión para una tienda online, abordando todas las fases del diseño e implementación de una base de datos relacional profesional:

- Modelado entidad-relación (E/R)
- Normalización hasta 3FN
- Creación de tablas con claves primarias y foráneas
- Inserción de datos reales y sintéticos (Mockaroo)
- Consultas SQL avanzadas
- Automatización con triggers
- Funciones y procedimientos en PL/pgSQL
- Gestión de roles y permisos
- Integración completa con Python
- Documentación técnica generada con Sphinx
